# Purpur.yml 方块与物品配置

本文档介绍 `purpur.yml` 中与方块和物品相关的配置选项。

---

## 方块配置

### 木桶 (Barrel)

```yaml
settings:
  blocks:
    barrel:
      six-rows: false
```

| 配置项 | 说明 |
|--------|------|
| six-rows | 木桶是否有 6 行存储空间（54 格） |

启用后，木桶将拥有与大箱子相同的存储空间（54 格），而不是默认的 27 格。

### 末影箱 (Ender Chest)

```yaml
settings:
  blocks:
    ender_chest:
      six-rows: false
```

| 配置项 | 说明 |
|--------|------|
| six-rows | 末影箱是否有 6 行存储空间（54 格） |

启用后，末影箱将拥有 54 格存储空间，大大增加玩家的随身存储能力。

### 哭泣的黑曜石 (Crying Obsidian)

```yaml
settings:
  blocks:
    crying_obsidian:
      valid-for-portal-frame: false
```

| 配置项 | 说明 |
|--------|------|
| valid-for-portal-frame | 是否可以用于建造下界传送门框架 |

启用后，哭泣的黑曜石也可以用来建造下界传送门，增加建筑的装饰性选择。

### 蘑菇方块

```yaml
settings:
  blocks:
    disable-mushroom-block-spread: false
```

| 配置项 | 说明 |
|--------|------|
| disable-mushroom-block-spread | 禁用蘑菇方块蔓延 |

### 音符盒 (Note Block)

```yaml
settings:
  blocks:
    disable-note-block-updates: false
```

| 配置项 | 说明 |
|--------|------|
| disable-note-block-updates | 禁用音符盒更新 |

禁用后，音符盒不会响应红石信号变化，可以用于某些特殊的红石机械设计。

### 紫颂植物 (Chorus Plant)

```yaml
settings:
  blocks:
    disable-chorus-plant-updates: false
```

| 配置项 | 说明 |
|--------|------|
| disable-chorus-plant-updates | 禁用紫颂植物更新 |

## 世界设置中的方块配置

以下配置位于 `world-settings` 节点下：

### 床 (Bed)

```yaml
world-settings:
  default:
    blocks:
      bed:
        explode: true
        explode-on-villager-sleep: false
```

| 配置项 | 说明 |
|--------|------|
| explode | 在下界/末地使用床时是否爆炸 |
| explode-on-villager-sleep | 村民睡觉时床是否爆炸 |

### 营火 (Campfire)

```yaml
world-settings:
  default:
    blocks:
      campfire:
        lit-when-placed: true
```

| 配置项 | 说明 |
|--------|------|
| lit-when-placed | 放置时是否自动点燃 |

### 耕地 (Farmland)

```yaml
world-settings:
  default:
    blocks:
      farmland:
        bypass-mob-griefing: false
        gets-moist-from-below: false
        use-alpha-farmland: false
        disable-trampling: false
```

| 配置项 | 说明 |
|--------|------|
| bypass-mob-griefing | 是否绕过 mobGriefing 规则 |
| gets-moist-from-below | 是否可以从下方获取水分 |
| use-alpha-farmland | 使用 Alpha 版本的耕地机制 |
| disable-trampling | 禁用踩踏（防止耕地被踩坏） |

**实用场景**：`disable-trampling: true` 可以防止玩家和生物踩坏耕地。

### 海龟蛋 (Turtle Egg)

```yaml
world-settings:
  default:
    blocks:
      turtle_egg:
        bypass-mob-griefing: false
        random-tick-crack-chance: 500
        feather-fall-distance: 0.0
```

| 配置项 | 说明 |
|--------|------|
| bypass-mob-griefing | 是否绕过 mobGriefing 规则 |
| random-tick-crack-chance | 随机刻破裂概率 |
| feather-fall-distance | 缓降距离 |

### 海绵 (Sponge)

```yaml
world-settings:
  default:
    blocks:
      sponge:
        absorbs-lava: false
        absorption:
          area: 64
          radius: 6
```

| 配置项 | 说明 |
|--------|------|
| absorbs-lava | 是否可以吸收岩浆 |
| absorption.area | 吸收面积 |
| absorption.radius | 吸收半径 |

**趣味功能**：启用 `absorbs-lava` 后，海绵可以吸收岩浆！

### 刷怪笼 (Spawner)

```yaml
world-settings:
  default:
    blocks:
      spawner:
        deactivate-by-redstone: false
        fix-mc-238526: false
```

| 配置项 | 说明 |
|--------|------|
| deactivate-by-redstone | 是否可以用红石信号禁用刷怪笼 |
| fix-mc-238526 | 修复 MC-238526 漏洞 |

**实用场景**：`deactivate-by-redstone: true` 允许玩家用红石控制刷怪笼的开关。

### 信标 (Beacon)

world-settings:
  default:

| 配置项 | 说明 |
|--------|------|
| effect-range.level-* | 各等级信标的效果范围 |

### 蜂巢 (Beehive)

## 物品配置

### 末影珍珠 (Ender Pearl)

```yaml
world-settings:
  default:
    gameplay-mechanics:
      item:
        ender-pearl:
          damage: 5
          cooldown: 20
          creative-cooldown: 20
          endermite-spawn-chance: 0.05
```

| 配置项 | 说明 |
|--------|------|
| damage | 使用末影珍珠造成的伤害 |
| cooldown | 冷却时间（游戏刻） |
| creative-cooldown | 创造模式冷却时间 |
| endermite-spawn-chance | 生成末影螨的概率 |

### 免疫物品

```yaml
world-settings:
  default:
    gameplay-mechanics:
      item:
        immune:
          cactus: []
          explosion: []
          fire: []
          lightning: []
```

| 配置项 | 说明 |
|--------|------|
| cactus | 免疫仙人掌伤害的物品列表 |
| explosion | 免疫爆炸伤害的物品列表 |
| fire | 免疫火焰伤害的物品列表 |
| lightning | 免疫闪电伤害的物品列表 |

**示例**：让下界合金物品免疫火焰

```yaml
immune:
  fire:
    - minecraft:netherite_sword
    - minecraft:netherite_pickaxe
    - minecraft:netherite_axe
    - minecraft:netherite_shovel
    - minecraft:netherite_hoe
```

### 物品帧 (Item Frame)

```yaml
world-settings:
  default:
    gameplay-mechanics:
      item:
        item-frame:
          damage-multiplier: 1.0
```

| 配置项 | 说明 |
|--------|------|
| damage-multiplier | 伤害倍数 |

---

## 注意事项

:::tip 提示
- 方块配置分为全局配置（`settings.blocks`）和世界配置（`world-settings.*.blocks`）
- 全局配置影响所有世界，世界配置可以覆盖全局配置
- 某些配置可能需要重启服务器才能生效
:::

:::warning 注意
- 修改存储空间（如 six-rows）可能影响现有物品
- 建议在修改前备份世界数据
- 某些配置可能影响游戏平衡
:::
